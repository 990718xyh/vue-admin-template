# vue-admin-template

 简体中文 | [English](./README.md)

## 开始
```sh
# 代码克隆 
$ git clone https://github.com/990718xyh/vue-admin-template.git
# 依赖下载
$ pnpm
# 开发环境启动
$ pnpm dev
```

## 发布
```sh
# 测试环境打包
$ pnpm build:test
# 线上环境打包
$ pnpm build:prod
```

## Advanced
```sh
# 预览
$ pnpm preivew
# eslint 代码校验
$ pnpm lint
# 格式化代码
$ pnpm prettier
```
